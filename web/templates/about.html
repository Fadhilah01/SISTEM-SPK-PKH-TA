Penjelasan untuk Sidang
Kenapa SVM?

SVM dipilih karena: (1) performa baik pada dataset dengan jumlah fitur sedang, (2) mampu menangani data non-linear dengan kernel trick, (3) memiliki dasar matematis yang kuat (mencari hyperplane optimal). Dibanding SAW/TOPSIS (SPK konvensional), SVM belajar dari data historis sehingga lebih objektif dan konsisten.

Kenapa Kernel RBF?

Data PKH memiliki hubungan non-linear antar variabel — tidak bisa dipisahkan dengan garis lurus. RBF memetakan data ke dimensi lebih tinggi sehingga hyperplane bisa memisahkan kelas Layak vs Tidak Layak dengan lebih baik. Dari 4 kernel yang diuji, RBF memberikan F1-Score tertinggi (0.89).

Arti Confusion Matrix

True Positive (TP): Diprediksi Layak, benar layak.
True Negative (TN): Diprediksi Tidak Layak, benar tidak layak.
False Positive (FP): Diprediksi Layak tapi tidak layak (kesalahan Tipe I).
False Negative (FN): Diprediksi Tidak Layak tapi layak (kesalahan Tipe II).
Model kita punya FN=12, FP=10 dari 200 data uji — cukup baik.

C=10 dan gamma=0.1 Artinya Apa?

C=10: Tingkat regularisasi sedang — tidak terlalu strict (overfit) maupun terlalu longgar (underfit).
gamma=0.1: Pengaruh satu sampel training cukup luas — model fokus ke pola umum, bukan detail noise. Nilai ini ditemukan melalui GridSearchCV dengan 5-fold cross validation.
