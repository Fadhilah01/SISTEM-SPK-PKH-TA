Proses import:
  1. Sistem akan membaca dan memvalidasi setiap baris data.
  2. Baris yang valid akan langsung disimpan ke database.
  3. Setiap data yang berhasil di-import akan otomatis diprediksi kelayakannya oleh model SVM.
  4. Baris yang gagal validasi akan dilewati dan ditampilkan daftar errornya.
  5. Import akan menambah data baru — duplikasi dicegah secara otomatis berdasarkan NIK.

Persyaratan File:

  • Baris pertama HARUS header (nama kolom)
  • Kolom wajib: nama, nik, alamat, penghasilan, pekerjaan, kepemilikan_aset
  • Kolom opsional wilayah: provinsi, kabupaten, kecamatan, desa_kelurahan
  • Kolom boolean diisi: YA/TIDAK, 1/0, Ada/Tidak
  • Kategori harus sesuai dengan yang didaftarkan sistem

Yang Terjadi Setelah Import:

  • Data tersimpan di database
  • Skor ordinal dihitung otomatis
  • Model SVM RBF memprediksi kelayakan
  • Hasil (Layak/Tidak) tersimpan
  • Siap dilihat di Dashboard & Data Calon
